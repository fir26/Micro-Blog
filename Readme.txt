Projet MicroBlog - VERLINGUE Baptiste

Le projet consiste a reproduire un site comportant un fil de message avec les fonctionnalites suivantes:

-Inscription sur le site
-Connexion sur le site
-Pouvoir envoyer un message
-Pouvoir le modifier
-Pouvoir le supprimer
-N'etre capable de realiser les 3 actions precedentes si et seulement si l'utilisateur est connecte
